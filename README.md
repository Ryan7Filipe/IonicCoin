# IonicCoin

Aplicativo de conversão de moedas desenvolvido com Ionic e Angular.

## Descrição

O IonicCoin é um aplicativo que permite converter valores entre diferentes moedas utilizando taxas de câmbio atualizadas. O sistema também mantém um histórico das conversões realizadas, armazenando os dados localmente.

## Funcionalidades

- Conversão de moedas
- Seleção de moeda de origem e destino
- Conversão inversa de moedas
- Histórico de conversões
- Armazenamento local com Local Storage
- Interface responsiva

## Tecnologias Utilizadas

- Ionic Framework
- Angular
- TypeScript
- HttpClient
- Local Storage

## Como Executar

1. Instale as dependências:

```bash
npm install
```

2. Execute o projeto:

```bash
ionic serve
```

## Estrutura do Projeto

- Home: tela principal de conversão.
- Histórico: exibe as conversões realizadas.
- Configurações: configurações do aplicativo.
